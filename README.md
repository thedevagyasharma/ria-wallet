# Ria Wallet

A React Native / Expo prototype for the Ria Money Transfer wallet experience.

## Accessing the preview

The app is distributed via **Expo Go** using EAS Update. No build or install from source is required.

### Steps

1. Install **Expo Go** on your device
   - [iOS — App Store](https://apps.apple.com/app/expo-go/id982107779)
   - [Android — Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. Open Expo Go and sign in (or create a free Expo account at [expo.dev](https://expo.dev))

3. Open this link on your device or scan the QR code below:

   ```
   exp://u.expo.dev/53d660c2-8a41-4adb-bf7c-40fb7ba20ec4?channel-name=main
   ```

   Or open the latest update directly from the EAS dashboard:
   [expo.dev/accounts/devagyasharma/projects/ria-wallet](https://expo.dev/accounts/devagyasharma/projects/ria-wallet/updates)

4. The app will load automatically. Future updates are delivered over-the-air — just close and reopen to get the latest version.

### Requirements

- iOS 16+ or Android 10+
- Expo Go app (latest version recommended)
- Internet connection for initial load
